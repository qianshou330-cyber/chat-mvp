## 必测路径

| 路径 | 通过标准 | 结果 |
| --- | --- | --- |
| 注册/登录 | 新账号可注册，退出后可重新登录 |  |
| 个人资料 | 昵称、简介、头像可保存并刷新后保留 |  |
| 好友申请 | A 申请 B，B 同意后才出现单聊；拒绝不创建单聊 |  |
| 单聊 | A/B 可互发文字消息，刷新后历史仍在 |  |
| 群聊 | 可建群、发群消息、查看成员 |  |
| 附件 | 允许类型且小于 10 MB 可上传，成员可打开 |  |
| 搜索 | 可搜联系人名、群名、中文消息关键词 |  |
| 通知 | B 开启通知后，A 发消息，B 收到通用提醒 |  |
| 权限 | C 看不到 A/B 私聊、附件或 push subscription |  |
| 移动端 | 主要页面无明显溢出或按钮不可点 |  |

## 每日检查

- 新 Issue 全部分配严重程度。
- P0 当天处理或停止外测。
- P1 记录复现步骤和修复计划。
- P2/P3 进入 `docs/beta-feedback-summary.md` 汇总。
- 修复后必须跑 `npm run lint`、`npm run test -- --run`、`npm run build`，再做生产 smoke test。

## v0.3 启动门槛

- 至少 3 名测试者完成必测路径。
- 至少 10 条反馈完成分类。
- P0 为 0。
- P1 不阻塞公司试用，且有明确处理计划。
- 非成员权限回归通过。
