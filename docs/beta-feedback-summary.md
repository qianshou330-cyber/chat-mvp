## v0.3 建议

默认顺序，除非外测反馈改变优先级：

1. 团队/工作区：支持一个小公司空间，成员在空间内协作。
2. 管理员成员管理：邀请、查看、移除成员，并明确权限影响。
3. 同账号多端登录管理：设备列表、退出其他设备、在线状态聚合。
4. 运营能力：错误日志、反馈入口、测试数据清理流程和生产 smoke test 清单。
5. 安全文档：隐私说明、数据保留说明、敏感信息提示和管理员使用说明。
