# v0.8.1-beta 公开公测上线清单

`v0.8.1-beta` 是 2026-07-01 约 30 人公开公测候选版。它不新增聊天大功能，重点收口 UI 质感、页面内提示、公开注册风险、生产 smoke 和上线后 72 小时巡检。

## 上线前 48 小时

- 确认 GitHub Actions `lint/test/build/e2e` 全绿。
- 确认 Vercel Production 加载最新 `main` bundle，`/manifest.webmanifest` 和 `/sw.js` 返回 200。
- 用 `codex.v081.*` 临时账号跑 A/B/C smoke：注册、登录、好友申请、单聊、群聊、群禁言、媒体、搜索、通知、登录设备、群管理记录。
- 用 Chrome 做真实 Web Push：B 开启通知，A 发消息，B 收到通用通知，payload 不包含正文、附件名或邮箱。
- 375px、390px、430px 视口检查聊天首页、聊天页、群详情页、个人资料页，确保无明显拥挤、溢出或后台感大卡片。
- 更新并检查公测文案：公开注册开放，但页面和邀请文案提醒不要发送敏感信息。

## 上线当天

- 发布 `v0.8.1-beta` tag 后再对外发链接。
- 公测链接：`https://chat-mvp-tau.vercel.app`
- 第一小时观察：注册数量、登录失败、消息失败、附件失败、通知失败、搜索失败、群管理失败。
- 所有反馈进入 GitHub Issues，按 P0/P1/P2/P3 标记。
- 若出现 P0，暂停扩大传播，保留现场记录，走最小修复或 Vercel 回滚。

## 上线后 72 小时

- 每天检查 Supabase `app_error_events` 和管理员运行状态，不记录消息正文、搜索关键词、附件名、token 或 secret。
- 每天确认 Storage 使用量、Edge Function/Webhook 通知状态、Vercel 部署状态。
- 每天抽查 C 类非成员权限：不能读取或搜索私聊、群消息、附件、群文件、push subscription 或管理记录。
- P1 必须有复现步骤、处理决策和负责人；P2/P3 先收敛到下一轮体验补丁。

## 不做事项

- 不做访问码或白名单；本轮接受公开注册风险。
- 不做频道、机器人、语音/视频通话、端到端加密、原生 App。
- 不引入专用搜索服务或自建后端。
- 不提交 service role、VAPID private key、webhook secret 或数据库密码。
