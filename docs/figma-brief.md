# Figma Brief: Chat App MVP - Mobile PWA

Create a mobile-first PWA design for a Telegram-style chat MVP. Keep the product familiar, fast, and clean without copying Telegram brand assets, logo, proprietary icons, or exact visual identity.

## Screens
- Splash: app mark, product name, loading state.
- Login: email field, passwordless sign-in action, setup notice.
- Chat list: profile entry, search, quick actions, recent conversations, unread counts.
- Chat window: compact header, message feed, attachment button, text composer, send button.
- Group info: group avatar, member count, add member, permissions, member list.
- Profile settings: avatar, name, bio, email, save, sign out.

## Style
- Mobile frame: 430 x 844.
- Primary color: #2aabee.
- Backgrounds: #ffffff, #f5f8fb, #e9f1f7.
- Accent support: green #2f9f72, amber #d49124, rose #d85c74, slate #4f657c.
- Radius: 8px maximum for rectangular UI.
- Typography: system sans, compact hierarchy, no oversized marketing hero.

## UX Notes
- Prioritize thumb reach and fast scanning.
- Keep chat rows stable at 74px height.
- Keep composer fixed at bottom.
- Avoid explanatory feature text inside app screens.
- Use common icons for search, send, attach, settings, back, sign out.
