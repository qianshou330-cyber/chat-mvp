# v0.2 Beta 外测邀请包

## 邀请文案

你好，我正在测试一个小范围 beta 版本的 Chat MVP。它是一个移动优先的 Web 聊天应用。

请用这个地址体验：

https://chat-mvp-tau.vercel.app

这只是小范围测试，不是公开发布。请暂时不要在里面发送敏感或隐私信息。当前版本还不支持端到端加密、语音/视频通话或原生手机 App。当前产品路线图不包含频道和机器人。

请帮忙测试这些流程：

1. 用邮箱和密码创建账号。
2. 退出登录后重新登录。
3. 修改显示名称、个人简介和头像。
4. 通过邮箱向另一个已注册测试用户发送好友申请。
5. 同意好友申请后发送单聊文字消息。
6. 创建一个群聊并发送群消息。
7. 上传一个 10 MB 以内的允许类型附件，并让另一名成员打开。
8. 搜索联系人名、群名和消息关键词。
9. 如果浏览器支持，开启消息通知，并让另一名测试者发消息确认能收到提醒。
10. 告诉我任何坏掉、看不懂、卡住、慢或者缺失的地方。

反馈入口：

https://github.com/qianshou330-cyber/chat-mvp/issues

请选择：

- 缺陷反馈：功能坏掉或流程被阻塞。
- 体验反馈：看不懂、不顺手、太慢、布局不舒服。
- 功能建议：后续版本想要的能力。
