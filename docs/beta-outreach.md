# Chat MVP 公开公测邀请包

## 邀请文案

你好，我正在做 Chat MVP 的公开公测。这是一个移动优先的 Web 聊天应用，当前目标是验证 30 人左右的轻量聊天、群管理、媒体、搜索和通知体验。

体验地址：

https://chat-mvp-tau.vercel.app

请注意：这是公测版，不是正式企业合规通讯产品。请不要在里面发送密码、密钥、客户资料、未公开财务信息、源码凭据或其他敏感内容。当前版本还不支持端到端加密、语音/视频通话或原生手机 App。当前产品路线图不包含频道和机器人。

请帮忙测试这些流程：

1. 用邮箱和密码创建账号。
2. 退出登录后重新登录。
3. 修改昵称、简介、图片头像或视频头像。
4. 通过邮箱向另一个已注册用户发送好友申请。
5. 同意好友申请后发送单聊文字消息。
6. 创建或加入群聊，发送群消息。
7. 上传图片、视频或 10 MB 以内的允许类型文件，并让另一名成员打开。
8. 搜索联系人名、群名、中文消息关键词，并尝试类型/发送人/时间筛选。
9. 如果浏览器支持，开启消息通知，并让另一名测试者发消息确认能收到提醒。
10. 告诉我任何坏掉、看不懂、卡住、慢或者布局不舒服的地方。

反馈入口：

https://github.com/qianshou330-cyber/chat-mvp/issues

请选择：

- 缺陷反馈：功能坏掉或流程被阻塞。
- 体验反馈：看不懂、不顺手、太慢、布局不舒服。
- 功能建议：后续版本想要的能力。
