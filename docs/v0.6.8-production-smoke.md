# v0.6.8-beta Production Smoke and Trial Launch Gate

## Summary

`v0.6.8-beta` is the final production-trial launch gate after `v0.6.7-beta`. It does not add product features, database migrations, Supabase policies, Edge Functions, or environment variables.

This gate is meant to prove the production stack with real Supabase/Vercel state before inviting 20-30 company trial users.

## Trial Launch Checklist

Before inviting real users:

- Pick one owner/admin as the operator.
- Ask every participant to register at `https://chat-mvp-tau.vercel.app`.
- Owner/admin adds registered users from group details by email.
- Send the administrator invitation copy below to trial members.
- Share `docs/company-trial-safety.md` before asking people to use the app.
- Collect all feedback through GitHub Issues and classify P0/P1/P2/P3.
- P0/P1 fixes stay minimal: stability, permissions, notifications, attachments, member management, or mobile blocking issues only.
